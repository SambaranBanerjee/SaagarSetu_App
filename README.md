# 🌊 SaagarSetu 

> **Smart India Hackathon (SIH) 2025 Project** 🏆

SaagarSetu is a cross-platform mobile application designed to empower coastal communities and authorities. It acts as a bridge to crowdsource real-time data on ocean hazards—such as abnormal tides, pollution, or marine life strandings—and provides a streamlined reporting mechanism to instantly alert relevant government bodies for swift action.

## 🚀 Key Features

- **🚨 Real-Time Hazard Reporting:** Quickly report ocean-related hazards with descriptions, image uploads, and severity tagging.
- **📍 Precise Geolocation:** Automatically captures and embeds accurate GPS coordinates for every incident reported.
- **👥 Community Crowdsourcing Feed:** View verified hazard reports from other users in your coastal vicinity to stay informed.
- **🏛️ Government Alert System:** Formats and dispatches critical, actionable data directly to local and national marine authorities.
- **📱 Cross-Platform Accessibility:** A smooth, highly responsive native experience optimized for both iOS and Android.

## ⚙️ Installation & Local Setup

Follow these instructions to get a copy of the project up and running on your local machine for development and testing.

### Prerequisites
- [Node.js](https://nodejs.org/) installed
- React Native environment setup (Android Studio for Android, Xcode for iOS)

### Steps

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/SambaranBanerjee/SaagarSetu_App.git](https://github.com/SambaranBanerjee/SaagarSetu_App.git)
   ```

2. **Navigate to the project directory:**
   ```bash
   cd SaagarSetu_App/SaagarSetu
   ```

3. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

4. **Start the application:**
   
   *For Android:*
   ```bash
   npm run android
   # or
   npx react-native run-android
   ```
   
   *For iOS:*
   ```bash
   npm run ios
   # or
   npx react-native run-ios
   ```
